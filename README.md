# docker-snorkel
snorkel tutorial image

# Usage
* Build & run: `docker-compose -f docker-compose.yml up`
* Open http://localhost:8888/ in your browser.
* Tutorial: https://github.com/HazyResearch/snorkel#learning-how-to-use-snorkel
